# sc2-coop-overlay

Always-on-top StarCraft II Co-op overlay.

Current status:
- Minimal overlay window (frameless + always on top)
- Simulated clock (temporary)
- Global quit hotkey: Ctrl + Shift + Q

Next goals:
- Read in-game mission timer (MM:SS) from the HUD
- Load mission timing data (JSON) derived from Co-op map triggers
- Show next attack wave/objective/bonus + warning countdowns + direction arrows
